# Font Files

Place font files here for different languages:

Required fonts:
- `Inter-Regular.ttf` - Default font for Latin scripts
- `NotoSansKR-Regular.ttf` - Korean
- `NotoSansArabic-Regular.ttf` - Arabic
- `NotoSansSC-Regular.ttf` - Simplified Chinese
- `NotoSansJP-Regular.ttf` - Japanese
- `Roboto-Regular.ttf` - Alternative font

Download from:
- Inter: https://fonts.google.com/specimen/Inter
- Noto Sans: https://fonts.google.com/noto
- Roboto: https://fonts.google.com/specimen/Roboto
